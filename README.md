# Things I Love About You

A small interactive web project created as a personal an emotional gift, built with **HTML, CSS and Vanilla JavaScript**.

This project displays meaningful reasons, one at a time, each paired with a short story revealed through a modal - designed to feel intimate, simple and special.

---

# Live Demo

[View the live site] (https://eunicelgar.github.io/things-about-you/)

---

## Features

- Sequential navigation (Next / Previous)
- Dynamic counter: *Cosa X de XX*
- Modal with personal anecdotes
- Smooth animations and transitions
- Responsive and mobile-friendly design

---

## Built with

- HTML5
- CSS3
- JavaScript Vanilla
- GitHub Pages for deployment

---

## What I Practiced

- DOM manipulation
- Event handling
- State management without frameworks
- UI/UX design for emotional storytelling
- Clean separation of data, logic, and presentation
